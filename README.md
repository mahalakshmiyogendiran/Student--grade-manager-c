# Student Grade Manager 📊

A C program that manages student grades and calculates results.

## What It Does
- Takes student name and marks for multiple subjects
- Calculates total marks and average percentage
- Assigns grades from A+ (Outstanding) to F (Fail)

## How To Run
1. Open any C compiler (onlinegdb.com)
2. Paste the code from grade_manager.c
3. Click Run and enter student details

## Skills Used
- C Programming
- Arrays and Loops
- Conditional Statements
- Formatted Input/Output
